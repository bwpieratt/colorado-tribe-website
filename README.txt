COLORADO TRIBE WEBSITE STARTER KIT

FILES
- index.html: Home page
- schedule.html: Complete 2026 schedule
- schedule-content.html: Schedule-only markup for embedding
- roster.html, stats.html, host-families.html, about.html, contact.html: Starter pages
- css/tribe.css: Shared styling
- js/tribe.js: Shared navigation and schedule interactions
- images/logos/: Team logos

GODADDY SETUP
1. Unzip the folder.
2. Upload every file and folder while preserving the folder structure.
3. For Linux/cPanel hosting, upload the contents into public_html.
4. Make sure index.html is in the public_html root.
5. Your schedule URL will be yourdomain.com/schedule.html.

IMPORTANT
- The GameChanger buttons still need a real GameChanger team or game URL.
- Open schedule.html in a code editor and search for data-url="".
- Paste the GameChanger URL between the quotation marks.
- Directions buttons already open Google Maps, Apple Maps, and Waze.
- Edit navigation links in each HTML page if GoDaddy uses a different page structure.

EMBED OPTION
If you must use GoDaddy Website Builder rather than file hosting:
- Paste schedule-content.html into an HTML/Embed section.
- The page must also load css/tribe.css and js/tribe.js.
- Some Website Builder plans may restrict external CSS or JavaScript.
