HOW TO PUBLISH
1. Unzip this package.
2. Open GitHub Desktop.
3. Choose Repository > Show in Finder.
4. Copy every file and folder from this package into the repository folder.
5. Choose Replace when macOS asks.
6. In GitHub Desktop, use summary:
   Add coaches news and sponsors pages
7. Click Commit to main.
8. Click Push origin.
9. Cloudflare deploys automatically.
